===============================================================================
MONITORING QUERIES
===============================================================================

1. Find employees who didn't checkout:
```sql
SELECT u.name, a.check_in, a.date
FROM attendance a
JOIN users u ON a.employee_id = u.id
WHERE a.check_out IS NULL
AND a.date = CURRENT_DATE;
```

2. Find admin-marked attendance:
```sql
SELECT u.name, a.date, a.status, a.admin_reason
FROM attendance a
JOIN users u ON a.employee_id = u.id
WHERE a.admin_marked = true
ORDER BY a.date DESC
LIMIT 20;
```

3. Find absent employees today:
```sql
SELECT u.name, u.email
FROM users u
LEFT JOIN attendance a ON u.id = a.employee_id AND a.date = CURRENT_DATE
WHERE u.role = 'employee'
AND u.is_active = true
AND (a.id IS NULL OR a.status = 'absent');
```

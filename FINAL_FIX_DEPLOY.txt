CHECK-IN RULES:
- Before 9:05 AM → attendance_value = 1.0 (Present)

CHECKOUT RULES:

===============================================================================
WHAT HAPPENS ON CHECKOUT
===============================================================================

1. User clicks "Start Checkout Process"
2. Enters checkout mode
3. Camera opens (front-facing)
4. User captures selfie
5. GPS automatically triggered
6. Location acquired
7. Both selfie and GPS required
8. User clicks "Check Out"
9. System calculates IST time
10. If before 5:30 PM → Apply penalty
11. Update attendance_value
12. Store checkout GPS in gps_data.checkout
13. Update check_out timestamp
14. Return success

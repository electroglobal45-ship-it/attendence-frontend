🚀 HOW TO USE
══════════════════════════════════════════════════════════════

1. NAVIGATE TO ANY BOARD
   - From sidebar, click "Boards" or "Projects"
   - Select any board

2. SWITCH VIEWS
   - Look for view switcher bar below board header
   - Click any view icon: Board | Table | Calendar | Timeline | Dashboard | Map

3. CREATE TASK FROM CALENDAR
   - Switch to Calendar view
   - Click any date in the calendar
   - Task detail modal opens automatically
   - Edit title, description, labels, members, etc.
   - Task appears on that date immediately

4. VIEW TASKS IN CALENDAR
   - Tasks shown as colored chips on their due dates
   - Color indicates priority
   - Click task chip to view/edit details
   - Navigate months with Prev/Next or Today button

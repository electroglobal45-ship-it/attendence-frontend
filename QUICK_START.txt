TROUBLESHOOTING
===============

Problem: Backend won't start
Solution:
- Check if another process is using port 5000
- Run: netstat -ano | findstr :5000
- Kill process if needed
- Try different port in .env file

Problem: Frontend won't start
Solution:
- Check if port 3000 is in use
- Close other Next.js instances
- Delete .next folder and restart
- Run: npm install (if packages missing)

Problem: Cards snap back after moving
Solution:
- Check backend is running
- Check browser console for errors
- Verify auth token in localStorage
- Check network tab for failed requests

Problem: No real-time updates
Solution:
- Wait 30 seconds (polling interval)
- Check browser console for errors
- Verify both users are on same board
- Refresh page if polling stopped

Problem: Camera won't open
Solution:
- Allow camera permissions
- Use HTTPS on mobile (or localhost)
- Check camera not used by other app
- Try different browser

COMMON ERRORS & FIXES
=====================

Error: "Cannot find module"
Fix: Run `npm install` in that directory

Error: "Port already in use"
Fix: Kill process on that port or change port

Error: "CORS error"
Fix: Check CORS_ORIGIN in backend/.env matches frontend URL

Error: "Unauthorized 401"
Fix: Login again, check auth token

Error: "Network request failed"
Fix: Check backend is running on port 5000

Common Questions:

Q: Why 30-second polling instead of WebSocket?
A: Simpler implementation, good enough for most use cases,

Q: What if optimistic update fails?

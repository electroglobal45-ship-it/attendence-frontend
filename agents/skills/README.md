# Agent Skills Index

This directory contains auto-generated skill documentation from agent task executions.

## Overview

As agents complete tasks, they automatically:
1. Analyze successful execution patterns
2. Extract reusable techniques and approaches
3. Generate skill documentation in Markdown format
4. Update this index with new skills

## Skills by Agent

### Claude Agent
Claude specializes in complex reasoning, architecture design, and code review.

### GPT Agent
GPT excels at code generation, refactoring, and documentation creation.

### Gemini Agent
Gemini focuses on data analysis, testing, and performance optimization.

## Skill File Format

Each skill file contains:
- **Description**: What the skill enables
- **Original Task**: The task that generated this skill
- **Patterns Identified**: Reusable patterns discovered
- **Examples**: Concrete examples of application
- **Metrics**: Performance and success metrics
- **When to Use**: Guidelines for skill application

## Continuous Learning

The system automatically:
- Updates existing skills with new examples
- Combines related skills into more powerful capabilities
- Increases confidence scores based on successful applications
- Prunes rarely-used or low-success skills

===============================================================================
ADMIN MANUAL MARKING
===============================================================================

NEW API ENDPOINT: POST /api/admin/mark-attendance

REQUEST BODY:
```json
{
  "employeeId": "uuid",
  "date": "2026-05-19",
  "action": "absent" | "half_day" | "mark_checkout",
  "reason": "Optional reason"
}
```

ACTIONS:

1. MARK ABSENT:
   - Creates or updates attendance record
   - Sets status = 'absent'
   - Sets attendance_value = 0
   - Adds admin_marked = true
   - Stores admin_reason

2. MARK HALF DAY:
   - Updates existing record (requires check-in)
   - Sets status = 'half_day'
   - Sets attendance_value = 0.5
   - Adds admin_marked = true
   - Stores admin_reason

3. MARK CHECKOUT:
   - Updates existing record (requires check-in)
   - Sets check_out = current time
   - Adds admin_marked = true
   - Stores admin_reason

EXAMPLE USAGE:
```javascript
// Mark employee as absent
fetch('/api/admin/mark-attendance', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer ' + token
  },
  body: JSON.stringify({
    employeeId: 'employee-uuid',
    date: '2026-05-19',
    action: 'absent',
    reason: 'Did not come to office'
  })
})

// Mark checkout for employee who forgot
fetch('/api/admin/mark-attendance', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer ' + token
  },
  body: JSON.stringify({
    employeeId: 'employee-uuid',
    date: '2026-05-19',
    action: 'mark_checkout',
    reason: 'Forgot to checkout'
  })
})
```

```sql
SELECT u.name, a.check_in, a.date
FROM attendance a
JOIN users u ON a.employee_id = u.id
WHERE a.check_out IS NULL
AND a.date = CURRENT_DATE
ORDER BY a.check_in;
```

2. Find admin-marked attendance:
```sql
SELECT u.name, a.date, a.status, a.admin_reason
FROM attendance a
JOIN users u ON a.employee_id = u.id
WHERE a.admin_marked = true
ORDER BY a.date DESC
LIMIT 50;
```

3. Find employees absent today:
```sql
SELECT u.name, u.email
FROM users u
LEFT JOIN attendance a ON u.id = a.employee_id AND a.date = CURRENT_DATE
WHERE u.role = 'employee'
AND u.is_active = true
AND (a.id IS NULL OR a.status = 'absent');
```

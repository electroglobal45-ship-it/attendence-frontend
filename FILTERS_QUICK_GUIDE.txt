═══════════════════════════════════════════════════════════════
  FILTER LOGIC
═══════════════════════════════════════════════════════════════

Example 1: Show all bugs assigned to John
→ Check Member: John
→ Check Label: Bug
→ Result: Only John's bug tasks

Example 2: Show overdue tasks from Dev board
→ Check Board: Development
→ Check Due Date: Overdue
→ Result: Only overdue Dev board tasks

Example 3: Show tasks with no due date
→ Check Due Date: No due date
→ Result: All tasks without due dates

ALL FILTERS USE "AND":
Board ✓ AND Member ✓ AND Label ✓ AND Date ✓
= Only tasks matching ALL criteria

HOW IT WORKS NOW:
-----------------
1. User clicks on label in TaskDetailModal
2. toggleLabel() immediately updates state
3. API PUT request sent to /api/v1/tasks/:taskId with labels array
4. Backend saves labels to tasks.labels column (JSONB)
5. Labels persist across page refreshes
6. Labels appear in task detail modal

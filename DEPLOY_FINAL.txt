===============================================================================
TROUBLESHOOTING
===============================================================================

IF CHECKOUT FAILS:
- Check if database migration ran successfully
- Verify checkout_selfie_url column exists
- Check browser console for errors
- Verify GPS permissions granted

IF HOLIDAY BLOCKING NOT WORKING:
- Check if holidays table has data
- Verify is_active = true for holidays
- Check browser console for API errors
- Verify date format is YYYY-MM-DD

IF SIDEBAR STILL SCROLLS:
- Clear browser cache
- Hard refresh (Ctrl+Shift+R)
- Check if h-screen class applied
- Test on different device

IF CAMERA NOT WORKING:
- Ensure HTTPS (required for camera)
- Check camera permissions
- Try different browser
- Test on real device (not emulator)
